# Matrizes
São variáveis multidimensionais que guardam diversos valores. Possuem linhas (vertical) e colunas(horizontal).

## Exemplo:
NOTAS POR TRIMESTRE<br>
Aluno 1 {7, 2, 8}<br>
Aluno 2 {5, 2 ,0}<br>
Aluno 3 {7, 7, 8}<br>
